# SenseBox Mini

SenseBox Mini is a compact ESP32-based room monitoring device that measures temperature, humidity, and air quality, and provides a local web dashboard for configuration and alerts.

## Features
- ESP32-C3 based
- Temperature, humidity, and air quality monitoring
- Local WiFi web interface
- Configurable alert thresholds
- Visual status LED and sound alerts
